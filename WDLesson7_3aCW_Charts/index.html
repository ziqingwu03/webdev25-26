Charts

This classwork challenges students to generate charts using the c3.js JavaScript library. c3 library is built using the d3.js JavaScript library, hence both libraries must be loaded. There are many types of charts with various customizations that can be applied to each chart. We will focus on Line, Bar and Pie charts with basic features.

Subject Analysis

Scenario: Create a chart that analyzes average grades of Math, English, Science and Social Studies courses by period.

Budget

Scenario: Create a chart that displays the breakup of your budget by percentages using a pie chart.
