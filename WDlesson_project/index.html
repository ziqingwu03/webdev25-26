Tree enviornment data

Welcome to the NYC Open Data website. In this project we used nyc open tree data collected includes tree species, diameter and perception of health and turn it into info cards and charts where people can easy find infomation about a specific tree and turning in to charts where people can use as data

In this website, we have an web page about info card about different trees and an webpage for charts. his project focuses on organizing and presenting tree-related environmental data from NYC Open Data in a clear and accessible format. Using charts, visualizations, and information cards, the project highlights key insights and trends within the dataset, making it easier to explore and understand the information. The goal is to transform raw data into meaningful visual summaries that support data exploration and informed decision-making.
